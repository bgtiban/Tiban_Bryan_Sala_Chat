# Sala de chat UDP
Aplicación java con dos programas main, uno para el servidor y otro para el cliente,
ambos en Java.

# Tecnologías usadas
- Java versión 8
- Swing

# Ejecución en local
- Abrir proyecto con IDE favorito, se recomienda IntelliJ
- Ejecutar el main de la clase Server configurando los argumentos deseados,
  por ejemplo: `127.0.0.1 9097 9098`
- Ejecutar el main de la clase Client configurando los argumentos deseados,
  por ejemplo: `127.0.0.1 9097 9098`.
- Ejecutar tantos clientes como se desee.
